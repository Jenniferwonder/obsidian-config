## 目标问题目的优先
>时间、精力有限！  
>用时间去实现最有价值的事！  
>尽可能利用工具，解决 **执行** 层面问题，拒绝重复或低效劳动。

| 任务                 |优先级  | 分工 |
| -------------------- | ---- | ------ |
| 明确目标、问题、目的 |   🟥   |   👩     |
| 产品设计             |  🟧    |   👩     |
| 执行                 |  🟨    |   🤖     |
| 整合 > 提炼 > 总结 > 复用                     |  🟩    |   👩     |
