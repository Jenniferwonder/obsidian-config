## 如何生成文件目录到 Markdown 文档
#### 1. 命令行输入：
- 打开对应文件夹终端
- 在终端输入 `tree > text.md`，即可在新的.md 文档中生成文件目录
#### 2. VS Code 插件：
- 下载后在对应文件夹位置，单击右键，选择“Generate to Tree”即可一键生成
