# Obs-主题样式美化

## 如何快速配置主题样式？

### 选择主题 🎨

可快速设置主题样式

#### 1. 简洁主题推荐——Minimal

#### 2. 高级主题推荐——Blue Topaz
#### 3. Viridian

### 下载必备插件 ⬇️

#### 1. Minimal Theme Settings

- 可调整 Minimal 主题样式

#### 2. Style Settings

- 可自定义样式

#### 3. Contextual Typography

- 可自定义文本样式

### 其他样式插件（择需下载）⬇️

#### 1. Better Word Count

- 可显示笔记字数或选中片段字数

#### 2. Hide Sidebars on Window Resize

- 可在窗口缩小时隐藏边栏

#### 3. Better CodeBlock

- 可在阅读视图，显示代码**标题**

#### 4. CodeBlock Enhancer

- 调整代码块样式，显示代码**行数**

#### 5. Folder Icon

- 为文件夹添加 Icon

#### 6. List Callout

色块列表突出问题

#### 7. Multi-Column Markdown/ Blue-topaz Admonition

- 分栏链接相关笔记
- 但 Admonition 中链接的笔记不会显示在当前笔记关联笔记中
