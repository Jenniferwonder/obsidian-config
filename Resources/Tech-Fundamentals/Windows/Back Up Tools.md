## 重装系统
1. 换新机软件安装：
- **Git: ** mkdir >git init > git remote upstream> git pull/ git clone > ... 
- **收藏夹**：可通过 Microsoft Edge, Chrome 浏览器账户同步功能 **同步**
- **VS Code **配置：通过 Sync 与 Git **同步**
- 其他软件：可通过 **同步空间** ~~备份~~，一件同步安装 (VS Code/ Notion/ 语雀/ 网盘/ Ocam/ Adobe/ Chrome/ Wechat)
