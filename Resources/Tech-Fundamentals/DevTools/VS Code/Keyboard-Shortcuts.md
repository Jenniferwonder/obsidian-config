## Windows Keyboard Shortcuts

Powertoys Run: `alt`+`space`  
Settings: `win`+`I`  
Switch between programs: `alt`+`tab`  
Switch between desktop and program: `win`+`D`  
Switch between windows in browser: `ctrl`+`tab`  
Create a virtual desktop: `ctrl`+`win`+`D`  
Switch between desktops: `ctrl`+`win`+`<`/`>`  
Put window to the left/ right: `win`+`<`/`>`  
Search: `win`+ `s`  
Rename folder: `fn`+`F2`  
Old start menu: `windows`+`x`

#### 3. Basic Editing

‚Ä¢ CTRL + C to Copy  
 ‚Ä¢ CTRL + V to Paste  
 ‚Ä¢ CTRL + X to Cut  
 ‚Ä¢ CTRL + Z to Undo  
 ‚Ä¢ CTRL + Y to Redo  
 ‚Ä¢ CTRL + D to Delete  
 ‚Ä¢ Alt + Shift + Up/Down Arrow to move bullets up and down

#### 4. Find and Search

‚Ä¢ CTRL + F to find text  
 ‚Ä¢ Windows Key + Q or Windows Key or Windows Key + S to search or just press the windows key

#### 5. Selection

‚Ä¢ CTRL + A to Select all  
 ‚Ä¢ CTRL + Click to select multiple individual items  
 ‚Ä¢ Shift + Click to select multiple items  
 ‚Ä¢ Select entire word: Shift + CTRL + Arrow  
 ‚Ä¢ Select text: `shift`+`<`/`>`  
 ‚Ä¢ Select to the end of a sentence: `shift`+`end`  
 ‚Ä¢ Select to the start of a sentence: `shift`+`home`

#### 6. Windows Management

‚Ä¢ Alt + Tab to jump between open windows  
 ‚Ä¢ Alt + F4/ `ctrl`+`W` to close a window  
 ‚Ä¢ Windows + # key to bring up app on task bar where the # corresponds with the position on the taskbar  
 ‚Ä¢ Windows Key + arrow keys to snap a window in different positions  
 ‚Ä¢ Windows + D to see desktop. Toggle back and forth  
 ‚Ä¢ Windows + M to minimize all windows

#### 7. Browser Navigation

‚Ä¢ Alt + left or Alt + right arrow to navigate forward or back  
 ‚Ä¢ CTRL + T to open a new tab  
 ‚Ä¢ CTRL + Shift + T to bring up due tab in browser  
 ‚Ä¢ CTRL + Tab to move to next tab  
 ‚Ä¢ CTRL + Shift + Tab to move to previous tab  
 ‚Ä¢ F5 or CTRL + R to refresh a web page

#### 8. Zoom

‚Ä¢ Windows + or Windows - to zoom in or out

#### 9. Virtual Desktop

‚Ä¢ Windows + Tab to open task view  
 ‚Ä¢ Windows + CTRL + D to add a virtual desktop  
 ‚Ä¢ Windows + CTRL + Left or Right Arrow Keys move between them  
 ‚Ä¢ Windows + CTRL + F4 to close virtual desktop

#### 10. Display

‚Ä¢ Windows + P to project to an external monitor

#### 11. Others

 ‚Ä¢ CTRL + P prints

 ‚Ä¢ Windows + E to launch File Explorer
