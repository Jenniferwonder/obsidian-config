## PKM-快速编辑输入
#### 1. Linter
- 可自动纠正文本样式（列表序号）
#### 2. ~~MetaEdit~~ (> Metadata Menu)
- `ALT`+`M`
- 可自动设置 Yaml 格式
- 配合 Linter，更新 Yaml 内容
#### 3. Metadata Menu（配合DB Folder/ Projects）
- 配置管理Metadata
- 自动提供可选填充项
#### 4. Number Heading
- 可自动为标题编号，可自定义编号样式
#### 5. Auto Link Title
- 可自动生成链接标题
#### 6. Multi-Column Markdown
- 可支持同一笔记多栏布局（文案 + 分镜概要）
#### 7. Hover Editor
- 支持编辑预览弹框
#### 8. Citation/ Footnote
- 快速添加脚注
